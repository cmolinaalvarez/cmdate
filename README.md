# cmdate

Una utilidad para manejar fechas en formatos timestamp y longtime.

# Intall

```bash

npm install cmdate

```

# Manual

https://platzi.com/cursos/nodejs/como-publicar-tu-primer-paquete-npm/

1. Crear un proyecto nuevo npm init -y
2. Ajustar el package.json
3. npm link desde el proyecto cmdate
4. crear el link desde el proyecto principal npm link cmdate  
5. Subir el proyecto al github
6. Subir el proyecto a npm https://www.npmjs.com/
